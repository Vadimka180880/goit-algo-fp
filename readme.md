# Проект: Моделювання кидання двох кубиків за допомогою методу Монте-Карло

## Завдання 7

### Опис
Цей проект реалізує метод Монте-Карло для моделювання кидання двох ігрових кубиків та побудови таблиці сум і їх ймовірностей. Метою є визначення ймовірності кожної можливої суми чисел, які випадають на двох кубиках, та порівняння отриманих результатів з аналітичними розрахунками.

### Реалізація

1. **Алгоритм симуляції кидків кубиків**:
   - Симулює велику кількість кидків двох кубиків.
   - Підраховує кількість кожної можливої суми (від 2 до 12).
   - Обчислює ймовірності кожної суми на основі симуляції.

2. **Алгоритм аналітичних розрахунків**:
   - Обчислює ймовірності кожної суми на основі відомих ймовірностей випадання кожної суми при киданні двох кубиків.

3. **Порівняння результатів**:
   - Порівнює ймовірності, отримані методом Монте-Карло, з аналітичними ймовірностями.
   - Створює графік для візуалізації результатів.

Результати симуляції методом Монте-Карло добре узгоджуються з аналітичними розрахунками, що підтверджує правильність реалізації симуляції. Невеликі відхилення в ймовірностях є нормальними для випадкових симуляцій, але загалом метод Монте-Карло показує надійні результати, близькі до аналітичних.
